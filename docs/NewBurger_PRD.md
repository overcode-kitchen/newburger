# 🍔 뉴버거 (NewBurger) 기획서 v1.0

> 국내 햄버거 신메뉴를 한눈에 모아보고, 유저 후기까지 남기는 반응형 웹 서비스  
> **MVP 목표: 1주 이내 출시 / 익명 서비스**

---

## 1. 서비스 핵심 정의

| 항목 | 내용 |
|------|------|
| 서비스명 | 뉴버거 (NewBurger) |
| 핵심 가치 | 흩어진 햄버거 신메뉴 정보를 한 곳에서 |
| 타겟 | 햄버거 좋아하는 20–30대 |
| 플랫폼 | 반응형 웹 (모바일 우선) |
| MVP 브랜드 | 맥도날드 / 버거킹 / 롯데리아 / 맘스터치 |

---

## 2. 페이지 구조

```
뉴버거
├── 홈 (신메뉴 피드)          → 메인 진입점
├── 메뉴 상세 (/menu/[id])    → 상세 정보 + 후기
└── 어드민 (/admin)           → 내부 전용 (비공개)
```

---

## 3. 메뉴 카드 정보 (홈 피드)

### 카드에 보여줄 것
- [ ] 메뉴 이미지
- [ ] 브랜드명 (맥도날드 / 버거킹 / 맘스터치)
- [ ] 메뉴명
- [ ] 가격
- [ ] 출시일
- [ ] 한정 뱃지 (판매기간 있는 경우)
- [ ] NEW 뱃지 (출시 2주 이내)
- [ ] 평균 별점

### 카드에 안 보여줄 것 (상세에서 보여줌)
- 칼로리
- 공식 링크
- 후기 내용

## 4. 메뉴 상세 페이지 정보

### 보여줄 것
- [ ] 메뉴 이미지 (크게)
- [ ] 브랜드명
- [ ] 메뉴명
- [ ] 가격
- [ ] 출시일
- [ ] 판매 기간 (한정 여부)
- [ ] 칼로리
- [ ] 공식 링크 버튼 → 브랜드 앱/사이트 연결
- [ ] 평균 별점 + 후기 수
- [ ] 후기 리스트 (최신순)

### 유저 액션
- [ ] 별점 선택 (★ 1–5)
- [ ] 후기 텍스트 작성 (선택)
- [ ] 공유하기 버튼 (링크 복사 / 카카오톡)

## 5. 탐색 방식

### MVP (Phase 1)
- [ ] 브랜드별 필터 탭 (전체 / 맥도날드 / 버거킹 / 롯데리아 / 맘스터치)
- [ ] 신메뉴순 정렬 (기본값)
- [ ] 인기순 정렬 (별점 높은 순)

### Phase 2
- [ ] 검색 기능
- [ ] 칼로리순 정렬
- [ ] 가격순 정렬

## 6. 유저 행동 범위

### MVP — 로그인 없이 익명으로
- [x] 별점 + 후기 남기기
- [x] 공유하기
- [ ] ~~즐겨찾기/북마크~~ → Phase 2 (로그인 필요)
- [ ] ~~먹어봤어요 체크~~ → Phase 2 (로그인 필요)

### Phase 2 — 로그인 도입 후
- [ ] 즐겨찾기 / 북마크
- [ ] 먹어봤어요 체크
- [ ] 내가 쓴 후기 모아보기
- [ ] 신메뉴 알림 구독

## 8. 데이터 구조

### Menu
```
- id
- brand (mcdonald / burgerking / lotteria / moms)
- name (메뉴명)
- price (가격)
- image_url
- release_date (출시일)
- end_date (판매 종료일 — null이면 상시)
- is_limited (한정 여부)
- calories (칼로리)
- official_link (공식 링크)
- description (설명)
```

### Review
```
- id
- menu_id
- rating (1–5)
- comment (텍스트 후기 — 선택)
- ip_hash (중복 방지용)
- created_at
```
